# 🐍 Python — Complete Learning Path

A structured, hands-on Python course designed to take you from **zero to ML/AI ready**.  
Each topic is in its own file with numbered sections, inline comments, and runnable examples.

---

## 📂 Topics

| #  | File | Topic | What You'll Learn |
|----|------|-------|-------------------|
| 01 | `01_variables.py` | Variables & Data Types | `str`, `int`, `float`, `bool`, type casting, arithmetic & assignment operators |
| 02 | `02_strings.py` | Strings | Indexing, slicing, f-strings, all string methods, escape characters |
| 03 | `03_conditionals.py` | Conditionals | `if/elif/else`, comparison & logical operators, ternary, `in`, `is` |
| 04 | `04_loops.py` | Loops | `for`, `while`, `range()`, `break`, `continue`, `enumerate()`, `zip()` |
| 05 | `05_list.py` | Lists | Indexing, slicing, list methods, list comprehension |
| 06 | `06_tuple.py` | Tuples | Immutability, packing/unpacking, `*rest` syntax |
| 07 | `07_set.py` | Sets | Union, intersection, difference, deduplication |
| 08 | `08_dictionary.py` | Dictionaries | Key-value pairs, looping, nesting, dict comprehension |
| 09 | `09_functions.py` | Functions | Parameters, `*args`, `**kwargs`, lambda, `map/filter/reduce`, scope |
| 10 | `10_oop.py` | OOP | Classes, inheritance, encapsulation, polymorphism, dunder methods |
| 11 | `11_error_handling.py` | Error Handling | `try/except/else/finally`, `raise`, custom exceptions |
| 12 | `12_file_handling.py` | File Handling | Read/write text, CSV, and JSON files |

---

## 🚀 How to Use

1. **Start from `01_variables.py`** and work your way down in order.
2. **Read the comments** — every line is explained.
3. **Run each file** to see the output:
   ```bash
   python3 01_variables.py
   ```
4. **Experiment** — change values, break things, and learn!

---

## 🎯 Goal

After completing all 12 files, you will have a solid Python foundation to move into:

- **NumPy** — Matrix math and tensors
- **Pandas** — Data manipulation (CSV, Excel, DataFrames)
- **Matplotlib / Seaborn** — Data visualization
- **Scikit-learn** — Classical Machine Learning
- **PyTorch / TensorFlow** — Deep Learning & AI
